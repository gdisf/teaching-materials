Backbone Exercise: Backend



Go through these steps, and do as many as you can in the time we have:

  • If you didn't finish the events exercise, download the starter files from here.
  • If you are not yet running your code from a local server, do that now by running python -m SimpleHTTPServer from your directory. We need to do this because of security restrictions on file:// protocol with XMLHttpRequests.
  • There is a JSON REST API at backchat-backend.appspot.com/messages that you can use from your app. Add the URL to the ChatMessages collection.
  • Use create instead of add
  • Try reloading index.html and sending a new chat message. Check NET requests to see if it sent to the server.
  • Change ChatMessagesView to fetch() the messages from the server, and stop passing in the test messages from app.js.
  • Add an event listener for the "reset" event on the collection that re-renders.
  • Try reloading index.html and confirm you see all the server messages.
  • Add an event listener to the refresh button to fetch new messages whenever its clicked. Reload and confirm that works (open in multiple tabs).
  • Fetch new messages continuously on a timer. Confirm that works.

Make sure that you use your browser developer tools to make debugging easier while working on this. Check for errors, and use console.log() to figure out how far your code makes it, and what the values of your variables are along the way.
