Backbone Exercise: Events



Go through these steps, and do as many as you can in the time we have:

  • If you didn't finish the views exercise, download the starter files from here.
  • In views.js, define a ChatFormView that renders the #chat-form-template.
  • In app.js, create a ChatFormView, pass in the chat messages collection, and render the view into the .chat-form-wrapper element.
  • Reload index.html and confirm that you see the form.
  • In the ChatFormView, add an event for the click event on the submit button that creates a new ChatMessage and adds it to the collection.
  • In the ChatMessagesView, add a listener on the collection for the 'add' event that re-renders the template.
  • Reload index.html, try sending a message, confirm that it shows up.

Make sure that you use your browser developer tools to make debugging easier while working on this. Check for errors, and use console.log() to figure out how far your code makes it, and what the values of your variables are along the way.
