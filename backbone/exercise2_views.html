Backbone Exercise: Views



Go through these steps, and do as many as you can in the time we have:

  • If you didn't finish the models exercise (up to step 6), download the starter files from here.
  • Create an app.js file and include it via a script tag in your HTML page. Make sure you're referencing it from the right location - check your console for 404s.
  • In app.js, create an array of test messages and create a ChatMessages collection out of them.
  • Create a views.js file and include it via a script tag in your HTML page.
  • In views.js, define a ChatMessagesView that renders the #chat-messages-template with the passed in collection.
  • In app.js, create a ChatMessagesView, and pass the collection into it. Render the view into the .chat-messages-wrapper element. Hint: The properties of the object that you pass to the template need to match the template variables. Look at the raw template in index.html.
  • Reload index.html and see if you see the messages.

Make sure that you use your browser developer tools to make debugging easier while working on this. Check for errors, and use console.log() to figure out how far your code makes it, and what the values of your variables are along the way.
