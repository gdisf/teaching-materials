Exercises: DOM Access

DOM Detective

Use DOM access methods to select

  • Open up girldevelopit.com in Chrome or Firefox, and open up the console.
  • Use DOM access methods to find the following parts of the page:
    • Every image on the page
    • The upper left-hand logo
    • The logos of the media sources that featured GirlDevelopIt
    • The big tagline in the middle
    • All of the green colored class names
    • The "CLASSES" text in the center area
    • The donate link


document.getElementsByTagName("img");
document.getElementById("logo");
document.getElementsByClassName("media");
document.getElementsByClassName("tagline")[0];
document.querySelectorAll("a.green");
document.querySelector(".blue .topic");
document.querySelector("#donate a");
