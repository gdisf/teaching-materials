Project: DOM Detective

The case of the GirlDevelopIt website

  • Open up girldevelopit.com in Chrome or Firefox, and open up the console (CMD-Option-I on Macs).
  • Use DOM access methods to find the following parts of the page. You probably want to use Inspect Element to figure out what their tag names, classes, and IDs are.
    1. Every image on the page
    2. The navigation area in the upper right
    3. The MailChimp sign-up form in the bottom
    4. The upper left-hand logo that says GDI (Hint: use CSS descendant selectors)
    5. The logos of the media sources that featured GirlDevelopIt (lifeHacker, TED, etc., at the bottom of the page)
    6. The big heading that says "DON'T BE SHY DEVELOP IT"
    7. BONUS: all the dots in the map


document.getElementsByTagName("img");
document.getElementsByClassName("nav");
document.getElementById("mc_embed_signup");
document.querySelector(".logo img");
document.querySelectorAll(".press-logos img");
document.querySelector(".opener h1");
document.querySelectorAll("circle");

The case of the BrainPickings blog

  • Open up brainpickings.com in Chrome or Firefox, and open up the console.
  • Use DOM access methods to find the following parts of the page. You probably want to use Inspect Element to figure out what their tag names, classes, and IDs are.
    1. The bylines of each blog post (the text that displays the author)
    2. Every image that is inside a blog post
    3. Every link inside the left-hand nav bar
    4. The Facebook link in the top right social media bar
    5. The input box for entering your name for email subscription, on the left side
    6. Every link inside the tag cloud on the left side
