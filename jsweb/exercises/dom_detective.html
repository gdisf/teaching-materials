Exercise: DOM Detective

CASE #1: Can She Develop It?

  • Navigate to theGirl Develop It site, and open up the Dev Tools console – Cmd-Opt-I (Mac), Ctrl-Shift-I (Windows)
  • Use the DOM access methods we learned to find the following parts of the page. There may be multiple ways to select the same DOM node.
    1. Every image on the page (as a collection)
    2. The navigation area in the upper right
    3. The MailChimp form at the bottom
    4. The GDI logo in the upper-left
    5. The media logos (TED, lifehacker, etc.), at the bottom
    6. The "DON'T BE SHY DEVELOP IT" headline
    7. All the red dots on the US map (as a collection)


      document.getElementsByTagName("img");
      document.getElementsByClassName("nav");
      document.getElementById("mc_embed_signup");
      document.querySelector(".logo img");
      document.querySelectorAll(".press-logos img");
      document.querySelector(".opener h1");
      document.querySelectorAll("circle");

CASE #2: Picking Brain Pickings

  • Open up brainpickings.com in Chrome or Firefox, and open up the console.
  • Use DOM access methods to find the following parts of the page. You probably first want to use Inspect Element to figure out what their tag names, classes, and IDs are.
    1. The bylines of each blog post, e.g. "BY MARIA POPOVA" (as a collection)
    2. Every image inside a blog post
    3. Every link inside the left nav bar
    4. The Instagram link in the left nav bar
    5. The "Name" input box for email subscription in the left nav bar
    6. Every quote insode a blog post
