jQuery Lesson Plan

An example meetup description for this workshop is here.


Facilitator Prep:


  • Create a class on a khanacademy.org account (like a shared one for your group).

  • Create the doc to share with students, like this one.

  • Make num_students printouts of Welcome doc (modifying the tinyurl and coach code).

  • Make (4*num_students/2) printouts of Pairing Time doc.

  • Invite students and TAs to the Slack channel.

  • Send reminders to students about reviewing material beforehand.



TA prep:


  • Try out the Khan Academy webpage environment and one of the jQuery challenges.

  • If you have any demo of a relevant project you’d like to share, let the facilitator know.


Schedule:


11:00

Setup tables, name tags

11:30

Students start arriving, eating lunch, setting up their laptops

Make sure they’ve added GDI chapter account as coach and that they’re on the Slack channel.

12:00

Welcome, start round of introductions with an intro quick question

12:10

Sponsor welcome, if a sponsor host is available

Present “What is GDI?”

12:15

Introduce yourself to person nearest to you, with an intro question

12:20

Talk about the structure for workshop - self-paced learning, *skip* the projects and keep going, then we pair on projects together.

12:25

Watch “Welcome to jQuery”

or Present Slides version (until "Why")

12:45

Start self-paced learning:

Tell them to spend no more than 15 minutes on reviewing material, so they can get started on jQuery.

1:00

Check-ins:

Walk around to each student. Check that they show up in your coach reports, that they're moving on from the review material, and that they're on the Slack channel.

2:25

Present What is pair programming?

(Give shortened version if most have already paired before)

2:35

Pair Project: (paste into doc)

If you’ve done DOM modification: Xtreme Makeover, Webpage Edition

Else If you’ve done DOM access: DOM Detective: girldevelopit.com

3:00

Project show & tell (if there are any)

3:05

Re-start self-paced learning

5:00

Pair Project: (paste into doc)

If you’ve done DOM events, do Project: Scene change

Else If you’ve done DOM modification: Xtreme Makeover, Webpage Edition

Else If you’ve done DOM access: DOM Detective: girldevelopit.com

6:10

Present “Thanks for coming!”

Go home!
