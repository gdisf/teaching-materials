Web Development Teaching Materials

Interested in teaching any of it?

  • Consult our teaching guide.
  • Fork the Github repository. Deploy it on a free App Engine account (which is how we host this version) or any static file server (even a public Dropbox folder could work).
  • Try doing the exercises yourself and make sure you understand what issues students might run into while doing them.
  • After you teach it, report any issues on the Github project or even better, send a pull request.
  • Let us know about what you taught where.
