Exercise 3: Audio and Video APIs

Let's make a synchronized audio/video watcher!

  • Start with your solution from the first exercise.
  • Add a "play" button to the HTML in the page.
  • Add JavaScript to the bottom of the page that adds a "click" event listener to that button. In the event callback, play both the audio and the video files.
  • Do the same for a pause button.
  • Add a "seeked" event listener to both the audio and video files. In the event callback, find out the currentTime of whichever file finished seeking, and set the other file to that time as well.
