Pancake Making Exercise


Write two algorithms, one that is space efficient, and one that is time efficient!


The function you write should accept one parameter: the number of servings to make.
The function should return an object with the following keys: time, and space.

  1. Pair up!
  2. Write pseudocode for this problem! Try to be as thorough and detailed as possible for each step.
  3. Walk through your pseudocode together with your partner using an example/ test case, to make sure each step is doing exactly what you think it should.
  4. Now turn your pseudocode into code! You can use any environment you want to run and test code.
    If you would like premade test cases in JavaScript, feel free to fork this repl.

Definitions and data structures
class Bowl {
  constructor(size) {
    this.size = size
  }

  fillWithBatter() { return 5 }
  whisk() { return 2 }
  refrigerate() { return 30 }
}

class Pancake {
  cook() { return 3 }
}
Stumped? Click here for some hints.
  1. Your space efficient algorithm should have constant space complexity - so a fixed size data structure (bowl).
  2. Your time efficient algorithm should have linear space complexity - so a data structure (bowl) whose size is dependent upon the input (servings).
