Exercise: Backbone



In this exercise, you will take the MVC app from the previous exercise and use Backbone for it instead.

Go through these steps, and do as many as you can in the time we have:

  • Either use your solution from the last site or download this file and save it into your project folder. Open it up in your browser and make sure it works. You should see a list of videos, and when you click on them, a modal pops up that plays the video.
  • Read through the code and try to understand what it's doing.
  • Add a script tag for underscore.js and backbone.js, using the URLs at cdnjs.com. Backbone depends on underscore, so the backbone script tag should be after the underscore one.

  • Modeling The Data

  • Replace the current VideoModel with a VideoModel object that extends Backbone.Model. It should have default values for the title and author, and it should set the embed and preview URL in the initialize method.
  • Replace the current VideoCollection with a VideoCollection object that extends Backbone.Collection. It should have a comparator based on the title property, so that the collection is sorted by title by default.
  • Modify the code in document.ready to create a VideoCollection.
  • Change the for loop to iterate through the items in the VideoCollection> and pass them to the VideoItemView.
  • Change your VideoItemView to use the attributes of the Backbone model via the get method.
  • Try loading the page now and see if your code works. Are there any errors? Look at the console and work through them one by one. You will have to modify a few things in your addVideoToList function.

  • Creating the View

  • Replace VideoItem with a view that extends Backbone.View. Start by putting all of the current render logic into the render function. The render function should append new elements to this.$el and return this. The code should reference this.model, the model that belongs to the view.
  • In the for loop in document.ready, change the code so that you pass in the video to VideoItem as the model. Call the render function and append the view's element to the global div.
  • Try loading the page and see if you can get the list of videos to display.
  • Now, add events to the view and make an event for click on the a element, which calls a renderModal function. Move the modal rendering code into that function.
  • Try loading the page now and see if you can get the modal to display when clicking the video.

Make sure that you use your browser developer tools to make debugging easier while working on this. Check for errors, and use console.log() to figure out how far your code makes it, and what the values of your variables are along the way.
