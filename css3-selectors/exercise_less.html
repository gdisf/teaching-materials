CSS3: Less Exercise

The Setup

You can skip this section if you are using Firefox.

If you are not yet running your code from a local server, do that now by following these instructions. We need to do this because of security restrictions on file:// protocol with XMLHttpRequests.

Coding

  1. Download this webpage into a folder. You'll be turning the CSS in that file into Less instead.
  2. Open the webpage and see what it looks like.
  3. Move the styles from the webpage into a separate file and save it as "styles.less".
  4. Add links to the LESS and JS file in the HEAD of your page: 
    <link rel="stylesheet/less" type="text/css" href="styles.less">
<script src="http://cdnjs.cloudflare.com/ajax/libs/less.js/2.7.2/less.min." type="text/javascript"></script>
  5. Open the page and check that it loads and is styled.
  6. Find all the colors used in the styles and turn them into variables. Less docs: variables
  7. Create a mix-in for rounded-corners and use that wherever border-radius is used. Less docs: mix-ins
  8. Create a mix-in for box-shadow and use that wherever box-shadow is used. Less docs: mix-ins (scroll)
  9. Re-structure the styles so that they are nested according to the DOM structure. For example, all of the sidebar related styles can nest underneath a main sidebar selector. Less docs: nested rules
